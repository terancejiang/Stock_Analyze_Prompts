# Stock Analyze Prompts

股票分析 AI Prompt 集合，策略灵感来源于B站UP主「史诗级韭菜」。

## 在线文档

[https://terancejiang.github.io/Stock_Analyze_Prompts/](https://terancejiang.github.io/Stock_Analyze_Prompts/)

## 目录结构

- `cigbutt/` — 烟蒂股分析 Prompt（多版本迭代）
- `turtle_framework/` — 龟龟投资策略 Prompt（多版本迭代）
- `reports/` — 基于以上 Prompt 生成的个股分析报告
